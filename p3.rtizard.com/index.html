P3 for Richard Tizard: In Page Biochemistry Tools

Currently many, many biochemistry and molecular biology web sites will perform routine calculations for you but most run on the server side. I propose to try two or more of these in a web page solely in Javascript. It is certainly possible that modern browsers running on modern hardware may be capable of adequate performance of some of these.

I'll flesh this out more in the near future, but two possible functions spring to mind.

1) www.isoelectric.ovh.org estimates the isoelectric point for a protein given a submitted peptide sequence. Input is text and output is a float. No graphical razzle dazzle. It runs this in PHP on the server. I propose doing this in Javascript right in the page. It will be an interesting test of the speed of this approach, since accepted methods involve looping through the amino acid sequence many times.

2) Mimicking web.expasy.org/translate would be computationally much easier. This function would accept a DNA sequence and output the protein sequence it encodes. One pass through looking up the triplets in a codon usage table. Input would be text and output also text. No graphics again, unless it were useful to highlight, say, one or more amino acids (say, all Cysteines) by Coloring them.

The interactions are admittedly minimal as outlined here. In each case a single input yields a single output. I will try to think of ways to augment this to exploit the interactivity of JS.
